Storymap
===================

Cette application permet de valoriser des données géolocalisées sous la forme de storymap. Il est possible d'enrichir avec du contenu édito.
Il existe pour le moment 2 templates (liste et carousel).

----------

Principe
-------------
Pour créer une nouvelle storymap, il suffit de créer un dossier dans le répertoire stories et d'y déposer les ressources nécessaires à savoir :

#### <i class="icon-file"></i> Création d'une storymap

> - un fichier json **config.json** (obligatoire)
> - un fichier css (optionnel)
> - un fichier csv (optionnel).
